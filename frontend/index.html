Scoring Logic

Weighted confidence with diversity and quality.

confidence = sum(weight × raw_signal_strength)
             × diversity_factor
             × mean(quality_scores)

The engine rewards multi-confirmation setups. Stronger raw events, better data quality, and more diverse event types all push the final confidence higher.
